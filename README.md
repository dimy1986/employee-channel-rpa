# 员工渠道RPA自动化机器人

## 项目概述

本项目是一个针对**桌面应用**"员工渠道"的自动化流程机器人，可自动执行日常的菜单导航、数据查询、报表导出等重复操作，帮助用户节省时间。

## 核心特性

✅ **多报表支持** - 支持10-20种不同类型的报表  
✅ **灵活查询** - 支持动态查询参数（如日期变化）  
✅ **多格式导出** - 支持Excel、PDF等多种格式  
✅ **浏览器弹窗处理** - 自动检测并处理浏览器弹窗场景  
✅ **错误通知** - 自动出错通知用户  
✅ **图像识别** - 结合UI定位和图像识别，提高稳定性  
✅ **日志记录** - 完整的执行日志和审计跟踪  

## 快速开始

### 1. 环境安装

```bash
pip install -r requirements.txt
```

### 2. UI录制

```bash
python src/utils/ui_locator.py record
```

### 3. 配置报表

编辑 `config/reports_config.yaml` 定义您的报表。

### 4. 运行机器人

```bash
python src/main_robot.py --all
```
